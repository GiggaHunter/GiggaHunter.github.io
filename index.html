Methodology

PHASE 01
Network Mapping
Identify all hosts: IP, MAC, hostname, OS, domain. Build complete picture of the environment.
NetworkMiner
PHASE 02
Protocol Analysis
Review Protocol Hierarchy for anomalies. Email protocols on a workstation = immediate red flag.
Wireshark
PHASE 03
Identify Infected Host
Filter suspicious protocols + Conversations view to pinpoint which host generates anomalous traffic.
Wireshark
PHASE 04
Infection Vector
HTTP first (unencrypted). HTTPS second — whitelist known good, focus on timing before C2 activity.
Wireshark
PHASE 05
C2 Analysis
Follow HTTP/TCP streams. Look for: fake User-Agent, encoded data, beaconing patterns, bot registration.
Wireshark
PHASE 06
IOC Verification
Verify IPs, domains, hashes on VirusTotal. PCAP is primary evidence — VT is secondary only.
VirusTotal
PHASE 07
Credentials
Extract leaked credentials: Kerberos hashes, NTLM challenges, HTTP cookies. Identify affected accounts.
NetworkMiner
COMING SOON
Malware Analysis
Static and dynamic analysis of extracted payloads. YARA rules, sandbox detonation, family identification.
